# Maze Explorer — Documentation

This directory contains the complete documentation for the **Maze Explorer** project.
The documentation has been divided into multiple specialized documents to clearly
separate concerns and improve readability.

Each document focuses on a specific aspect of the system, including architecture,
algorithms, simulation, visualization, and user interaction.

---

## Documentation Index

- **SETUP.md** — Developer setup and installation instructions  
- **USER_GUIDE.md** — End-user manual  
- **API.md** — Internal API and module interfaces  
- **ARCHITECTURE.md** — System architecture and design decisions  
- **SECURITY.md** — Safety, constraints, and correctness considerations  
- **REALTIME_CHAT.md** — Step-by-step simulation and event-driven execution  
- **DATABASE.md** — Data model and internal state representation  
- **DEPLOYMENT.md** — Execution, portability, and environment considerations  

---

## Intended Audience

This documentation is intended for:
- students and academic reviewers,
- software engineers,
- instructors evaluating algorithmic correctness,
- users interested in maze-solving algorithms.

The documents can be read independently or as a complete set.
